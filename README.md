# HTML Resume Page

This Assignment is a simple **HTML Resume Page built using only HTML** and deployed using **GitHub Pages**.

## Live Demo
https://sonasantosh.github.io/

## About the Project
- Created a **single-page resume**
- Built using **pure HTML**
- Followed **semantic HTML elements** for better structure, readability, and accessibility
- No CSS or JavaScript used (as per assignment requirement)

## Technologies Used
- HTML5 (Semantic HTML)

## Semantic Elements Used
- `<header>`
- `<main>`
- `<section>`
- `<h1>, <h2>, <h3>, <p>, etc`
- `<footer>`

## Deployment
The project is deployed using **GitHub Pages**.

## Purpose
This project was created as an **HTML-only assignment** to demonstrate:
- Proper HTML structure
- Use of semantic tags
- Clean and readable markup

## Note
Styling (CSS) and interactivity (JavaScript) are intentionally not included.
